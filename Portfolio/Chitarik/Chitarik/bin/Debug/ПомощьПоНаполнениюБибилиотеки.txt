--//////////////////[ Добавление категории ]/////////////////--
1. Зайти в папку с программой
2. Зайти в папку Lib
3. создать новую папку категории (имя категории не должно повторяться - не может быть нескольких категорий с одним и тем же именем)


--//////////////////[ обавление и удаление слов в категорию ]/////////////////--
1. Зайти в папку с программой
2. Зайти в папку Lib
3. Зайти в папку нужной категории (если нужно - создать новую папку категории)
4. Открыть файл с именем Слова (текстовый документ) - если файла нет, то надо создать его
5. Добавлять слова - каждое слово на новой строке (если слова уже были в файле - в конец документа)
6. На цыфры после слов не обращать внимание, если таакие есть в файле - это индексы ударения в слове
7. При необходимости любое слово можно удалить
8. Сохранять файл нужно в Unicode - (Файл->Сохранить как->Кодировка-> выбираем Unicode->жмем сохранить - если будет говорить, что файл уже существует - соглашаться и перезаписывать файл)


--//////////////////[ Добавление и удаление предложений в категорию ]/////////////////--
1. Выполнить действия до пункта 3 включительно в разделе [ Добавление и удаление слов в категорию ]
2. Открыть файл с именем Предложения (текстовый документ) - если файла нет, то надо создать его 
3. Добавлять предложения - обязательно наличие точки.
4. Сохранять файл нужно в Unicode - (Файл->Сохранить как->Кодировка-> выбираем Unicode->жмем сохранить - если будет говорить, что файл уже существует - соглашаться и перезаписывать файл)


--//////////////////[ Добавление и удаление стишков в категорию ]/////////////////--
1. Выполнить действия до пункта 3 включительно в разделе [ Добавление и удаление слов в категорию ]
2. Открыть папку с именем Стишки - если ее нет, то надо создать - не забываем: имя папки должно быть Стишки 
3. Для добавления стишка можно создать текстовый документ с названием стишка и в него вставить нужный стих и сохранить.
4. В каком виде будет вставлен стих (форматирование, перенос строк и слов) - в таком виде программа будет пытаться вывести его на экран, если пространство позволит.
5. Для удаления стиха - достаточно удалить текстовый документ с его названием
6. Сохранять файл нужно в Unicode - (Файл->Сохранить как->Кодировка-> выбираем Unicode->жмем сохранить - если будет говорить, что файл уже существует - соглашаться и перезаписывать файл)

--//////////////////[ Добавление и удаление скороговорок в категорию ]/////////////////--
1. Выполнить действия до пункта 3 включительно в разделе [ Добавление и удаление слов в категорию ]
2. Открыть файл с именем Скороговорки (текстовый документ) - если файла нет, то надо создать его
3. Добавлять скороговорки в нужном форматировании - между скороговорками нужно на новой строке вставляить разделитель @ (если в файле уже есть содержимое - добавлять в конец документа)
4. Сохранять файл нужно в Unicode - (Файл->Сохранить как->Кодировка-> выбираем Unicode->жмем сохранить - если будет говорить, что файл уже существует - соглашаться и перезаписывать файл)
