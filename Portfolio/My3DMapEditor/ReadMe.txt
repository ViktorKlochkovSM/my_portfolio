Для просмотра исходного кода(без компиляции) нужна VisualStudio 2005(8).
Для компиляции проекта в Vs 2005 нужен DX SDK и Framework 2.0.
Для запуска exe - фала программы нужен DirectX (версия 9.0 - желательно).
exe - файл, а также папки с готовыми картами и текстурами находятся в папке bin в корне проекта.

Техническое задание, руководство пользователя, план разработки программы, 
скриншоты и прочая дополнительная документация находятся в папке Documentation.

Для просмотра программы в действии просто запустите файл My3DMapEditor.exe
из папки bin.

В целях економии места в проекте оставлена одна готовая 3D-карта и ее текстуры.

Данная программа разрабатывалась и тестировалась на ПК:
WinXp
processor 3 GHZ
OZU - 768 Mb
Video - Nvidia 5.600 (258 Mb)
